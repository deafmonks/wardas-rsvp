Wardas RSVP

This repository contains a static RSVP form (`index.html`, `style.css`) and server helpers (proxy and example serverless code) to save RSVPs to Google Sheets.

How to push to GitHub:

1. Create a GitHub repo and copy the remote URL.
2. Run:

   ```bash
   git remote add origin <REMOTE_URL>
   git branch -M main
   git push -u origin main
   ```

Vercel deployment: connect this repo to Vercel or use the Vercel CLI to deploy.
